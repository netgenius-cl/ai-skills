# Integration Discovery

Leer esta referencia solo cuando falten integraciones o haya que ordenar la aclaracion.

## Lo minimo que debe venir claro

- sistema o app de origen
- sistema o app de destino
- evento que dispara
- resultado esperado

## Preguntas cortas sugeridas

- "Que apps quieres conectar?"
- "Que evento debe disparar esto?"
- "A donde debe terminar la informacion?"

## Regla

Estas aclaraciones idealmente las hace el orquestador antes de entrar de lleno a la skill.

Si no vienen y bloquean el trabajo:

- pedir una sola pregunta corta
- no abrir una entrevista larga
