# Access Modes

## Prioridad

1. ruta local sincronizada
2. carpeta montada en el sistema
3. enlace web si el entorno puede abrirlo
4. pedir export o ruta exacta

## Preguntas simples utiles

- "Pasame la ruta exacta de esa carpeta."
- "Lo tienes como carpeta en Windows o solo como enlace?"
- "Quieres que solo lea archivos o tambien que los ordene?"

## Respuestas cortas modelo

Si hay ruta local:

`Perfecto. Con esa carpeta lo puedo revisar directo.`

Si solo hay enlace y el entorno no puede abrirlo:

`Con solo ese enlace aqui no me alcanza. Necesito la carpeta sincronizada o el archivo exportado.`

Si falta una herramienta o permiso:

`Casi listo. Solo falta acceso a esa carpeta.`
