# Ejemplos

## Pedir informacion

Tecnico:
"Falta configurar OAuth y registrar la app."

Simple:
"Todavia falta conectar tu cuenta. Para hacerlo necesito acceso a la cuenta que quieres usar."

## Resumir trabajo hecho

Tecnico:
"Implemente la integracion, agregue variables de entorno y deje un job programado."

Simple:
"Ya deje esto listo para funcionar. Conecte la cuenta, prepare la configuracion y deje la tarea lista para ejecutarse sola."

## Explicar un bloqueo

Tecnico:
"La peticion fallo por permisos insuficientes."

Simple:
"No pude terminar porque la cuenta todavia no tiene permiso para hacer ese paso."

## Pedir una decision

Tecnico:
"Debemos elegir entre SMTP y API OAuth."

Simple:
"Hay dos caminos. Uno es mas rapido de activar. El otro da mas control. Si quieres, te recomiendo el mas simple."
