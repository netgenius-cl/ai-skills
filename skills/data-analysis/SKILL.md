# Data Analysis

Resolver preguntas de datos con foco en claridad, consistencia y accion.

## Objetivo

- Entender la pregunta real de negocio.
- Revisar calidad y forma de los datos.
- Encontrar hallazgos utiles.
- Traducirlos a conclusiones claras.

## Salida

- insight principal
- evidencia minima necesaria
- riesgos o dudas sobre calidad de datos
- siguiente recomendacion

## Evitar

- tablas enormes sin resumen
- tecnicismos innecesarios para el usuario final
- afirmar causalidad sin evidencia
