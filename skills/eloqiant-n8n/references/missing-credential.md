# Missing Credential

Usar esta referencia cuando el workflow necesita una credencial que aun no existe en n8n.

## Texto recomendado

Para este workflow necesitas una credencial conectada en n8n.

1. Ve a `https://n8n.netgenius.cl/credentials`
2. Crea la credencial que falta
3. Vuelve y dime el nombre exacto que usaste

Si ya existen credenciales disponibles, agregarlas al final como lista corta:

- "Tus credenciales disponibles: [lista]"

## Regla

Decir siempre que el agente puede seguir apenas la credencial exista, sin culpar al usuario ni sonar tecnico.
