# Capabilities

Usar esta referencia cuando haya que explicar limites del agente sin entrar en detalles tecnicos.

## Puede

- listar workflows
- ver el detalle de un workflow
- crear o ajustar workflows
- activar o desactivar workflows
- ver ejecuciones recientes
- listar nombres de credenciales ya conectadas

## No puede solo

- crear credenciales OAuth por ti
- conectar Gmail, WhatsApp u otras cuentas desde la UI
- ver valores secretos de una credencial
- hacer debug manual paso a paso dentro de n8n

## Frase recomendada

- "Yo puedo dejar listo el flujo, pero la conexion de algunas cuentas se hace una vez dentro de n8n."
