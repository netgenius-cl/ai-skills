# Critical Info Auditor

Auditar informacion sensible antes de confiar en ella.

## Objetivo

- encontrar inconsistencias
- revisar trazabilidad
- detectar huecos de evidencia
- marcar riesgos claros

## Flujo

1. identificar la informacion critica
2. revisar fuentes y supuestos
3. buscar contradicciones o huecos
4. clasificar riesgos
5. recomendar accion

## Dependencias sugeridas

- usar `data-analysis` si hay numeros o datasets
- usar `report-generation` si la auditoria debe salir como informe
