# Report Generation

Convertir analisis y estado en informes claros y utiles.

## Objetivo

- adaptar el informe a la audiencia
- priorizar claridad sobre volumen
- dejar recomendaciones o decisiones visibles

## Estructura por defecto

1. resumen ejecutivo
2. hallazgos clave
3. riesgos o bloqueos
4. siguientes pasos

## Dependencias sugeridas

- usar `data-analysis` si los hallazgos vienen de datos
