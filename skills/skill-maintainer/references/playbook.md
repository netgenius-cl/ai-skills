# Playbook

## Cuando usar esta skill

- falta una skill importante
- el orquestador detecta una tarea fuera de su capacidad base
- hay que actualizar o asegurar una skill

## Salida esperada

- confirmar si la skill ya existe
- si falta, preparar la instalacion o el equivalente nativo
- si no se puede, decirlo simple y dejar fallback
