# Codex

## Metodo nativo

Codex usa skills instalables desde GitHub.

## Instalar netgenius

```text
Usa $skill-installer e instala:
https://github.com/netgenius-cl/ai-skills/tree/main/skills/netgenius
```

Reinicia Codex si no aparece de inmediato.

## Skill de apoyo

```text
Usa $skill-installer e instala:
https://github.com/netgenius-cl/ai-skills/tree/main/skills/respuesta-simple
```

## Regla de uso

- instalar primero `netgenius`
- usar `respuesta-simple` solo como apoyo
